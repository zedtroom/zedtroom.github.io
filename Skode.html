Skode - Apprendre en ligne

Ce projet consistait à développer une application d'apprentissage en ligne pour une entreprise fictive. J'ai été responsable de la conception et du développement de l'application.

Objectifs :

Fournir une interface conviviale pour les clients pour effectuer des cours Gérer les réseaux en ligne Apprendre via notre site de manière intuitive

Compétences utilisées :

Conception d'interface utilisateur Développement d'applications (HTML, CSS, JavaScript) Promulger le site en ligne sur youtube Gestion de projet

Résultats : L'application a été livrée dans les délais impartis Les clients peuvent maintenant apprendre en ligne avec notre site Le site a bien été promulgé en ligne via des réseaux sociaux
